# 🛒 E-commerce Return Rate Reduction Analysis

## 📌 Project Overview
This project analyzes an online retail dataset to understand customer return behavior and identify products with a high return risk. The project includes data cleaning, exploratory data analysis (EDA), return analysis, machine learning, and a Power BI dashboard.

---

## 🎯 Objectives

- Clean and preprocess the retail dataset.
- Analyze sales and return trends.
- Identify top-selling and high-return products.
- Predict product returns using Logistic Regression.
- Build an interactive Power BI dashboard.

---
